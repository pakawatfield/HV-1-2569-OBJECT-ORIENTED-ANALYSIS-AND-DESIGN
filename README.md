# HV-1-2569-OBJECT-ORIENTED-ANALYSIS-AND-DESIGN
หลักการและขั้นตอนการพัฒนาระบบสารสนเทศ จัดทำเอกสาร (System Requirement Specification : SRS) บริหารจัดการในส่วนของวิเคราะห์และออกแบบระบบ พื้นฐานและแนวคิดเชิงวัตถุ กระบวนการวิเคราะห์และออกแบบเชิงวัตถุ โมเดลที่ใช้ออกแบบเชิงวัตถุ หลักการของ UML Modeling องค์ประกอบของ UML และการวิเคราะห์และออกแบบโปรแกรมทางธุรกิจ
